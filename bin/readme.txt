Ultima Online: Shattered Legacy Server (Emulator)
-------------------------------------------------

Version history:
----------------

- r5y2006 (16 April 2006): alot of core fixes,
                           added documentation and the two new procedures SetScriptName() and
                           PrimitiveSetScriptName().

- r4y2006 (16 April 2006): fixes again,
                           added new code: combat works better now (to test just set your melee
                           skill high in players.ini), death and resurrection with corpses and so on...

- r3y2006 (15 April 2006): added some scripting documentation,
                           commented scripts.

- r2y2006 (13 April 2006): fixed a bug concerning warmode,
                           removed "library" usage in scripts,
                           changed icon.

- r1y2006 (14 January 2006): fixed a small bug concerning status showing.

- r1y2005 (30 May 2005): first release,
                         probably there are a lot of bugs that couldn't be
                         found yet due to lack of testers,
                         still missing some documentation (for example
                         about events).


Development
-----------

This software was written in Delphi and uses the following libraries:

- "Innerfuse Pascal Script III" by Carlo Cock (http://www.carlo-kok.com)
- "Internet Component Suite" by François Piette (http://www.overbyte.be)
- "Hash Library" by Ciaran McCreesh
- "RichEdit Syntax Highlight" by Serhiy Perevoznyk (http://users.chello.be/ws36637)

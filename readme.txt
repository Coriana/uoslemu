This is the source distribution for the UO:SL server emulator intended to be used in the free shard named Ultima Online: A New Age.


Folders:
./bin - contains additional files not generated by compilation (documentation, scripts, data, etc.)
./lib - contains the external libraries used in the actual source
./src - contains the (Delphi 6) source code for the server emulator and the license text


Neither is the emulator an official Ultima Online server application nor is Ultima Online: A New Age an official Ultima Online shard or product.
